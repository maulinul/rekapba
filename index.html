          ⚙️ Setting

          "Otomatis" mengikuti setting gelap/terang di perangkatmu.
          Pilih "Jangan pisahkan" kalau baris paste-mu tidak memuat nama mandor, atau kamu mau mandor ditentukan dari dropdown saja.
          Ganti ke "Inggris" hanya kalau kamu copy angka dari Excel ber‑locale Inggris sehingga parse meleset.
          Sistem mencari kata ini di nama pekerjaan lalu mengambil 1 kata sesudahnya (cth "Dapur" → "Dapur Randusari"). Tambah istilahmu sendiri, misal nama proyek/gedung.
          Kalau diisi, parser memakai regex ini. Urutan grup (...) dari kiri: 3 grup = (persen)(kontrak)(kumulatif) · 4 grup = (pekerjaan)(persen)(kontrak)(kumulatif) · 5 grup = (mandor)(pekerjaan)(persen)(kontrak)(kumulatif). Persen tanpa %, uang tanpa "Rp".
          Token disimpan aman di localStorage browser lokal. Butuh scope gist (buat di GitHub Settings → Personal Access Tokens).
          ID Gist (contoh: 3a1f9b2...). Kosongkan saat Push pertama kali untuk membuat Gist rahasia baru secara otomatis.
